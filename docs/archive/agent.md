# agent.md

> 归档说明：本文件内容已合并进 `docs/AGENTS.md`，仅保留历史上下文。当前产品方向与开发约束请优先服从 `docs/AGENTS.md`。

## 当前项目方向

- 这是一个本地优先的 Android 智能记账 App。
- 自动识别结果必须先进草稿，不能直接作为正式流水。
- 只有已确认流水进入首页、图表、预算统计。
- OCR 失败、通知解析失败、截图识别失败都必须可见。
- 当前参考皮肤只保留 4 个底部页面：首页、账单、图表、设置。

## 当前账号与使用边界

- 未登录用户仍可使用本地记账、账单、图表、设置。
- 云同步与 AI 对话仅对已登录用户开放。
- 退出登录后必须清除认证状态并回到本地模式。

## 开发时优先遵守

1. 先读产品文档，再改代码。
2. 最小改动优先，不做与任务无关的重写。
3. 不为图省事破坏真实业务流程。
4. 不做 destructive migration。
5. UI 改动后同步更新文档。

## 验证重点

- `:app:assembleDebug` 能通过
- 草稿流程不被破坏
- 底部 4 页可正常切换
- 登录 / 退出登录状态正确
- 本地模式下仍可完成基础记账与查看
